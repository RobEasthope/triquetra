# Free Flight Academy

## Getting started

- Next dev: `yarn dev`
- Storybook: `yarn storybook`
- Sanity: `yarn sanity`

## Ignore build deployment rules

Next: `git diff --quiet HEAD^ HEAD ./`

Sanity: `git diff --quiet HEAD^ HEAD ./ ../next/*.schema.js ../next/utils`
